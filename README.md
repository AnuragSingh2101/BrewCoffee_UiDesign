## ✨ Features

- **☕ Specialty Coffee & Pastry Menu**:
  - Interactive tabbed categories: *Espresso Bar*, *Hand Brewed*, *Signature Drinks*, and *Pastries & Bites*.
  - Rich menu cards with high-resolution photography, badges (*Classic*, *Bestseller*, *Signature*, *Fresh Daily*), descriptions, and pricing.
- **🛒 Dynamic Slide-Out Shopping Cart Drawer**:
  - Live cart count badge in header.
  - Add items to order with automated tax (8%) and grand total calculations.
  - Quantity controls (`+` / `-`), item removal, and checkout confirmation toasts.
- **🎨 Premium Theme & High Contrast UI**:
  - Seamless adaptation across Light and Dark system preferences.
  - Deep espresso dark modes (`#18110F` background, `#2A1C19` card surface) with warm gold accents (`#D4AF37`) and crisp cream typography (`#F5F0E6`).
- **🖼️ Interactive Space Gallery & Lightbox**:
  - Full-screen lightbox modal for viewing cafe atmosphere, pour-over bar, and pastry counters.
- **💌 Interactive Newsletter & Notifications**:
  - Custom toast notification system for instant feedback on orders, subscriptions, and actions.
- **📱 Fully Responsive Design**:
  - Optimized for desktop, laptop, tablet, and mobile devices using CSS Grid and Flexbox.

## 🛠️ Technologies Used

- **HTML5**: Semantic elements, accessibility attributes (`aria-*`), and modern structure.
- **Vanilla CSS3**: CSS Custom Properties (Variables), CSS Grid, Flexbox, smooth transitions, and glassmorphism.
- **Vanilla JavaScript (ES6+)**: Event delegation, LocalStorage persistence, DOM manipulation, and state management.
- **Typography**: [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) (Serif) & [Poppins](https://fonts.google.com/specimen/Poppins) (Sans-serif) via Google Fonts.
